# PDF Merger

A simple and efficient tool to merge multiple PDF files into a single document.

## Overview
This project is a PDF Merger tool that allows users to combine multiple PDF files into a single document. It provides a user-friendly interface and leverages JavaScript, HTML, and CSS to deliver a seamless experience.

## Technologies Used
- **JavaScript (55.9%)**: Core logic for PDF merging and file handling
- **HTML (37.8%)**: Frontend structure and user interface
- **CSS (6.3%)**: Styling and visual design

## Features
- ✅ Merge multiple PDF files into one document
- ✅ User-friendly web interface
- ✅ Support for various PDF file sizes
- ✅ Progress indicator during the merging process
- ✅ Easy download of merged PDF files

## Installation
To get started with the PDF Merger project:

1. Clone the repository:
   ```bash
   git clone https://github.com/singhardhendra5-cpu/pdf-merger.git
   ```

2. Navigate into the project directory:
   ```bash
   cd pdf-merger
   ```

3. Open the `index.html` file in your web browser to start using the application.

## Usage
1. Open the application in your web browser
2. Click the upload button to select multiple PDF files
3. Click the 'Merge' button to start the merging process
4. Wait for the process to complete
5. Download the merged PDF file

## Project Structure
- `index.html` - Main entry point of the application
- `style.css` - Styling and layout
- `script.js` - JavaScript logic for PDF merging
